# 🌿 Seeds

Aplicativo mobile desenvolvido com React Native e TypeScript, focado em onboarding de usuários, verificação documental e descoberta personalizada de produtos em um ambiente de marketplace regulado.

---

## ✨ Sobre o projeto

O **Seeds** é um conceito de aplicativo mobile criado para simular a experiência de uma plataforma moderna que exige validação de identidade e análise documental antes de liberar acesso completo ao sistema.

O projeto foi pensado para demonstrar fluxos reais de produto, incluindo:

- Cadastro de usuários (Pessoa Física e Pessoa Jurídica)
- Upload e verificação de documentos
- Fluxo de aprovação de contas
- Navegação por catálogo de produtos
- Descoberta personalizada baseada no perfil do usuário
- Estrutura para e-commerce de acessórios
- Interface mobile moderna e intuitiva

---

## 📱 Funcionalidades

- 🔐 Autenticação de usuários (login e cadastro)
- 🧾 Envio de documentos para validação
- ⏳ Status de aprovação (em análise, aprovado, pendente, reprovado)
- 🏠 Tela inicial com recomendações
- 🔎 Exploração de catálogo com filtros
- 🛒 Carrinho de compras
- 📦 Gestão de pedidos
- 📍 Cadastro de endereços
- 📚 Seção educativa (tutorial e conteúdo)
- 👤 Área do usuário
